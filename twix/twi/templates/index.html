Twix with Django

Welcome to Twix with Django. This is a simple Django project that uses Tailwind CSS.

Features

  • Simple Django project
  • Uses Tailwind CSS
  • Responsive design

Requirements

  • Python 3.6+
  • Django 3.0+
  • Tailwind CSS

Installation

To install this project, follow these steps:


# Clone the repository
git clone

# Change directory
cd twix-with-django

# Install dependencies
pip install -r requirements.txt


# Run the server
python manage.py runserver
